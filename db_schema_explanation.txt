USERS
пользователи системы - id, email, пароль, имя, фамилия, активен(да/нет)
тестовый пользователь (ADMIN) admin@admin.ru, пароль 111

ROLES
список ролей Security - id, название роли

USERS_ROLES
таблица для связи многие-ко-многим между таблицами USERS и ROLES

PRODUCTS
товары в магазине - id, наименование, цена, удален(да/нет)

CATEGORIES
категории товаров - id, наименование категории

PRODUCTS_CATEGORIES
таблица для связи многие-ко-многим между таблицами PRODUCTS и CATEGORIES

CARTS
корзины покупателей (пользователей) - id, user_id.
Связь один-к-одному с таблицей USERS
Связь один-к-многим с таблицей CART_POSITIONS

CART_POSITIONS
товар в корзине - id, cart_id, product_id, количество товара
Связь многие-к-одному с таблицей CARTS
Связь многие-к-одному с таблицей PRODUCTS
